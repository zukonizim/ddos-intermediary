# ddos-intermediary
## ddos attack nedir?
İnternet trafiğinin yoğun olduğu bir ortamda hedefleri veya çevredeki altyapıyı kesen Dağıtılabilir Hizmet Reddi (DDOS) saldırı sunucusu

DDoS saldırıları, saldırı trafiği kaynağı olarak birden fazla ele geçirilmiş bilgisayar sistemini kullanarak etkili olur. Arama motorları, bilgisayarları ve IoT cihazları gibi diğer ağ kaynaklarını içerebilir.
Daha üst seviyeden bakıldığında DDOS saldırısı, otoyolda beklenmedik bir şekilde sıkışan ve düzenli trafiğin hedefine ulaşmasını engelleyen bir trafik sıkışıklığına benzer.
